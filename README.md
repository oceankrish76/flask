# Flask Webpage

A modern flask Application with routing, featuring Info and Services pages.

## Features

- Title and description on the homepage
- Navigation between pages
- Info page with company information
- Services page showcasing available services
- Modern, responsive UI design


Templates:
templates/base.html - Base template with title, description, and navigation
templates/index.html - Home page
templates/info.html - Info page
templates/services.html - Services page


## Installation

1. Install dependencies:
pip install flask
OR    pip install -r requirements.txt

2. Run the application:
python app.py

3. Open the browser and navigate to http://localhost:5000

4. Update documentation
5. Ssh key tested
